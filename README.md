## Description

This is API for a simple app to track all the job applications you made or offers you got during your job hunt. WIP

Frontend for the app can be found at: https://github.com/dawwidd/jobber-fe-nuxt

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
